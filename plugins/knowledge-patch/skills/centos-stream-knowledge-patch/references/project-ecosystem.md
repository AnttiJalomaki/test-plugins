# Project ecosystem

## Hosting and delivery

### Pagure repository migration

Pagure is being sunset. Migrate repositories hosted there to GitLab or Fedora Forgejo.

### RDO release-format migration

RDO plans to move future releases from RPM packages to Source-to-Image releases because packaging resources are limited. Consumers of RPM repositories should prepare for a different delivery and deployment workflow. Continued community RPM packaging is possible but not assured.
